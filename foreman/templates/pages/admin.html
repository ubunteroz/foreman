Evidence Retention

If you have a defined evidence retention period, Foreman can automatically send you an email to destroy evidence when that retention period comes up. Just select yes for reminders, and add the number of months to wait before the reminder is triggered. The user who added the evidence, as well as the current administrator(s) will get the email (in case that user is no longer an active Foreman user) X number of months after the date that the evidence was set to 'archived'. Emails will only be sent once, unless the number of months to trigger is changed below.

Note that currently archived evidence that has been archived longer than X months will have emails triggered when evidence retention is selected.

${forms.formSelectBox("Evidence Retention Reminders", "evi_ret", "evi_ret", "form_label", "input_long", evi_retention_options, please_select=True, first=True, value=options.evidence_retention)} ${forms.formTextField("Evidence Retention Period (months)", "evi_ret_months", "evi_ret_months", "form_label", "input_long", p_id="evi_ret_month_select", value=options.evidence_retention_period)} ${forms.formCheckBoxes("Remove existing reminders", "remove_evi_ret", "remove_evi_ret", "remove_evi_ret", "form_label", "radio_input_short", p_id="evi_ret_remove_exist")}
